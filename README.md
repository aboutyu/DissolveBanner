# THDissolveBanner
#### Super Easy Image Disolve.

# Feature
- [x] pure swift 5.0 code
- [x] require Kingfisher Library

# Installation

#### CocoaPods
Simply add AACarousel to your `Podfile`.

```
pod 'THDissolveBanner'
```
Enter command instruction into your terminal.

```
pod install
```
